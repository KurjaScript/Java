Dos命令相当于用指令操作一件事情

- Windows + R 输入cmd

- 或在任意文件夹下面，shift+右键，在此处打开命令行窗口
- 或资源管理器的地址栏前面加上cmd路径

### 常用的DOS命令

```bash
// 盘符切换 F:
// 查看当前目录下的所有文件 dir
// 切换目录 cd (change dir) 
cd /d f:\dir
cd ..  //返回上一级
cd 目录名字 // 进入同级目录
// 退出终端 exit

// 查看电脑ip  ipconfig

// ping 命令 ,用来测试网络是否正常
ping www.baidu.com

// 创建一个文件夹 md 文件夹
// 新建文件 cd>文件名.后缀
// 删除文件 del 文件名.后缀
// 移除文件夹 rd 文件夹
```
