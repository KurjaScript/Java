### 第一代语言

**机器语言**，二进制代码，直接输入给计算机使用，不经过任何转换

### 第二代语言

**汇编语言**

解决人类无法读懂机器语言的问题，用指令代替二进制

```
codesg segment
	start:
		mov ax, 0123h  ;  寄存器送入值
		mov bx, 0457h  
		add ax, bx     ;  ax + bx 寄存器值相加
		add ax, ax     ;  算 ax 的平方
	mov ax, 4o00h
	int 21h            ;  程序结束返回
	
codesg ends
end start              ;  指定程序入口标号为 start
```

**目前应用**

- 逆向工程
- 机器人
- 病毒
- ...
